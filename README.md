# Middle East Explore - Web Programming Assignment

## Theme
This website represents a fictional travel agency called **Middle East Explore**.  
The goal is to showcase travel packages, introduce the organization, and allow users to contact the agency.

The project uses ONLY HTML5, following the assignment requirements.

## File Organization

---

## What Each Page Contains

### **Home Page (index.html)**
- Header with navigation  
- Two articles  
- Images  
- Footer with author name  

### **About Page (about.html)**
- Section  
- Figure + figcaption  
- Semantic tags: nav, aside, mark, time, blockquote, details  

### **Services Page (services.html)**
- Table with:
  - 4 columns  
  - 5 rows  
  - thead, tbody, tfoot  

### **Contact Page (contact.html)**
- Form with:
  - fieldset + legend  
  - inputs (text, email, date)  
  - textarea  
  - select (4 options)  
  - submit button  

---

## Challenges Faced
- Maintaining consistent navigation across pages  
- Planning the structure using only HTML (no CSS allowed)  
- Organizing the table with proper semantic sections  
- Ensuring all pages link correctly with relative paths
